Website Pre Loader

This module allows Pre Loader when page takes time to load in the website. Pre Loaders are what you see on the screen while the rest of the page's content is still loading. Preloaders are important interface elements that let visitors know that the website hasn't crashed, it's just processing data and keep visitors entertained while server operations finish processing. Interesting animations can keep your users engaged while they're waiting for the page to load.
